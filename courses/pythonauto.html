Course Description

In this course, we will cover all of the basics of automation using python. This includes but is not limited to: web-scraping, controlling the mouse and keyboard, and sending automated emails and text messages. Learning automation really gives students the ability to create useful programs and harness the true power of software.

Who

Students Ages 12 or Above Who

  • Have an Intermediate Understanding of Python
  • Participated in Introduction to Python Course
  • Have had Previous Experiences with Python
